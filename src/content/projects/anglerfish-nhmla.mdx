Built a light sensor system for the exhibit of a rare Pacific footballfish specimen.

The tissue is highly sensitive to light. The ichthyology collection staff needed to limit and track light exposure.

**Publications:** <a target="_blank" href="https://www.popsci.com/animals/deep-sea-anglerfish-biofluorescence/">Popular Science</a>, <a target="_blank" href="https://www.smithsonianmag.com/smart-news/unusual-deep-sea-fish-washed-ashore-sandy-california-beach-180977721/">Smithsonian</a>, <a target="_blank" href="https://www.theguardian.com/us-news/2021/may/11/deep-sea-anglerfish-california-beach-finding-nemo">The Guardian</a>, <a target="_blank" href="https://www.newsweek.com/rare-deep-sea-fish-washes-california-beach-1590031">Newsweek</a>, <a target="_blank" href="https://www.cnn.com/2021/05/11/us/anglerfish-football-fish-california-trnd-scn/index.html">CNN</a>, <a target="_blank" href="https://www.latimes.com/california/story/2021-05-10/deep-sea-fish-footballfish-anglerfish-washes-up-newport-beach">LA Times</a>, <a target="_blank" href="https://www.atlasobscura.com/articles/deep-sea-footballfish">Atlas Obscura</a>
