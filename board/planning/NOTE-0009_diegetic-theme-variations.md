# Diegetic Theme Variations

## Idea

Extend the theme switcher beyond color palette changes into full **diegetic environment shifts**. Each theme reframes the site as a different narrative universe — not just a recolor, but a change in the portrait image of Grant (rendered in a different AI/illustrative style) and the thematic context of the entire surrounding environment (background textures, ambient effects, copy tone, section framing).

## Example Themes

- **Contained Specimen (SCP-style)** — Grant is an anomalous entity that has escaped containment. Redacted document styling, classification headers, warning labels, clipboard/file-folder chrome, security camera overlays.
- **Spy / Secret Agent** — Mid-century espionage aesthetic, dossier framing, mission briefing tone, typewriter fonts, classified stamps, surveillance motifs.
- **Dr. Manhattan–esque Energy Being** — Cosmic/quantum aesthetic, particle fields, deep blue glow, fractured time motifs, god-tier detached narration.
- (Open for more — cyberpunk operative, cryptid field guide entry, retro arcade boss, etc.)

## Notes

- This is a **long-term aspirational idea**, not near-term work. Flagged for future exploration once the base design system and color-theme switcher are mature.
- Would require: multiple styled portraits of Grant (AI-generated in each aesthetic), per-theme background systems, per-theme copy variants or framing chrome, and a theme switcher architecture that can swap more than CSS variables.
- Could become a signature "holy shit" moment of the portfolio — the kind of thing that makes hiring managers remember the site.
- Worth considering whether one "base" theme stays canonical and the others are unlockable easter-egg modes, versus presenting all themes as equal peers on the switcher.
